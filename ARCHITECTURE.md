# Project structure

```
kungfu/
├── .github/
│   ├── workflows/
│   │   ├── ci.yml              # PR + push CI (test/clippy/fmt/docs/js-binding)
│   │   └── benchmark.yml       # nightly benchmark job
│   ├── ISSUE_TEMPLATE/
│   │   ├── bug_report.md
│   │   ├── feature_request.md
│   │   └── perf_regression.md
│   └── PULL_REQUEST_TEMPLATE.md
├── bench/                      # comparison harness
│   ├── actix/                  # Rust — actix-web equivalent
│   ├── express/                # Node.js — Express equivalent
│   ├── fastapi/                # Python — FastAPI equivalent
│   └── results/                # generated by run-bench-suite.sh
├── bindings/
│   └── js/                     # napi-rs binding for Node.js + TypeScript
├── cli/                        # the `kungfu` CLI binary + kungfu_bench
├── core/                       # kungfu-core — the engine
│   └── src/
│       ├── server/
│       │   ├── mod.rs          # tokio + httparse, hand-rolled HTTP/1.1
│       │   ├── pool.rs         # buffer pooling (no per-request alloc)
│       │   ├── hot_reload.rs   # notify + atomic router swap
│       │   └── io_uring.rs     # V1 io_uring zero-copy path + pipelining
│       ├── router/             # trie with :params + *wildcards
│       ├── middleware/         # onion pipeline + built-ins
│       ├── openapi/            # auto OpenAPI 3.1 from route table
│       ├── request/            # Request type
│       ├── response/           # Response type + pool.rs (response recycling)
│       ├── headers.rs          # SmallVec-backed Headers type (V1)
│       ├── error.rs            # KungfuError { code, message, detail, suggestion }
│       └── version.rs
├── css/                        # kungfu-css — Tailwind-like utility engine
├── frontend/                   # SSR + live reload + TS type gen
├── kungfu/                     # idiomatic Rust API crate
│   └── examples/               # hello, middleware, params, errors, hot_reload
├── kungfu-macros/              # proc-macro crate (#[derive(Model)])
├── orm/                        # built-in ORM (mock + sqlx feature gates)
├── scripts/
│   ├── push-to-github.sh
│   ├── run-bench-suite.sh
│   ├── commit-file-by-file.sh
│   └── push-via-api.py
├── Cargo.toml                  # workspace root
├── Cargo.lock
├── README.md
├── PERF.md                     # perf engineering write-up
├── ROADMAP.md
├── CHANGELOG.md
├── CONTRIBUTING.md
├── CODE_OF_CONDUCT.md
├── SECURITY.md
├── LICENSE-MIT
└── LICENSE-APACHE
```
